# Files to Upload to GitHub

You need to upload these files to your GitHub repository:

## Required Files:
1. `package.json` (copy from local)
2. `server.js` (copy from local)
3. `tg-bot/index.js` (copy from local)
4. `brand/brand.js` (copy from local)

## Public Folder (copy all files):
5. `public/index.html`
6. `public/content-production.html`
7. `public/about.html`
8. `public/history.html`

---

## Don't upload:
- node_modules (too large)
- .env (contains secrets)
- Any log files
